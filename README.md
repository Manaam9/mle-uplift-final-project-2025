# Uplift Modeling Project

Проект посвящён построению и улучшению uplift-модели для оценки индивидуального эффекта маркетингового воздействия.

В рамках работы реализованы:

* baseline-модели (S-, T-, X-, R-learner);
* сравнение алгоритмов (CatBoost, RandomForest);
* подбор гиперпараметров (Optuna);
* анализ метрик (Uplift AUC, Qini AUC, Uplift@30);
* визуализация результатов;
* финальная модель для продакшена.

# Описание проекта

## Данные

Датасет содержит информацию о пользователях и маркетинговом воздействии:

* `treatment` — был ли применён маркетинговый стимул
* `target` — совершил ли пользователь целевое действие
* признаки поведения пользователей (`history`, `recency`, и др.)

---

## Подход

В проекте были реализованы и сравнены:

* **S-learner**
* **T-learner**
* **X-learner**
* **R-learner**

В качестве базовых алгоритмов использовались:

* `CatBoost`
* `RandomForest`

---

## Лучшая модель

По результатам экспериментов выбрана:

**R-learner + RandomForest (с оптимизацией гиперпараметров)**

Финальные метрики:

* **Uplift@30 ≈ 0.0549**
* **Uplift AUC ≈ 0.0229**
* **Qini AUC ≈ 0.0514**

Модель успешно выделяет пользователей, наиболее чувствительных к воздействию, и может использоваться для таргетирования.

---

# Метрики

В проекте используются следующие метрики:

* **Uplift AUC** — качество ранжирования по uplift
* **Qini AUC** — прирост относительно случайной модели
* **Uplift@30** — ключевая бизнес-метрика
